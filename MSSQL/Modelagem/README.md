# Modelagem

**SQL - MODELO FISICO**

```sql
CREATE TABLE ALUNO(
        IDALUNO INT PRIMARY KEY IDENTITY,
        NOME VARCHAR(30) NOT NULL,
        SEXO CHAR(1) NOT NULL,
        NASCIMENTO DATE NOT NULL,
        EMAIL VARCHAR(30) UNIQUE
)
GO

/* CONSTRAINTS */
 ALTER TABLE ALUNO
 ADD CONSTRAINT CK_SEXO CHECK (SEXO IN('M','F'))
 GO
```
